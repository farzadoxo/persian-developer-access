# این ریپوزیتوری چیه؟
توی شرایط قطعی اینترنت یکی از اقشاری که بیشترین آسیب رو میبینن توسعه دهنده ها و اصطلاحا IT Guy ها هستند. (البته که اینترنت حق همگانی هستش و نباید نیاز بهش بر اساس شغل و فعالیت های افراد دسته بندی بشه)

خوشبختانه سرویس های داخلی وجود دارن که در دوران قطعی اینترنت امکان دسترسی به اون ها فراهمه.توی این ریپوزیتوری لیست کاملی از این سرویس ها هستش که میتونید استفاده کنید.
(این سرویس ها شامل میرور ها,مستندات,ابزار ها و... هستند)

# نحوه مشارکت:
این ریپوزیتوری با مشارکت شما رشد خواهد کرد.این وظیفه انسانی ماست که در مواجهه با شرایط قطعی اینترنت موجب ایجاد و حفظ جریان اطلاعات آزاد علمی باشیم.
مراحل مشارکت به صورت زیر هست:

1. ابتدا این ریپو رو فورک کنید
2. طبق موضوع سرویس,آدرس اون رو توی فایل مارک دان مربوطه (طبق قالب موجود) اضافه کنید
3. تغییرات رو کامیت و پوش کنید
4. در نهایت یه Pull request باز کنید

(میتونید قالب افزودن آدرس رو [اینجا](https://github.com/farzadoxo/persian-developer-access/blob/master/template.md) ببینید)
